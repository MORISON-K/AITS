Academic Issue Tracker API

Welcome to the Academic Issue Tracker API. This REST API provides endpoints for managing academic issues, comments, and notifications.

API Endpoints

  • Authentication: /api/token/ - Get JWT token
  • Refresh Token: /api/token/refresh/ - Refresh JWT token
  • Registration: /api/register/ - Register new user
  • User Profile: /api/profile/ - User profile operations
  • Current User: /api/current-user/ - Get current user info
  • Issues: /api/issues/ - Issue operations
  • Comments: /api/comments/ - Comment operations
  • Notifications: /api/notifications/ - Notification operations
  • Issue History: /api/issuehistory/ - Issue history operations
  • Faculty Members: /api/faculty-members/ - Get faculty members
  • System Info: /api/system-info/ - Get system information

API Status

The API is running and ready to accept requests. For more information, check out the API documentation or use a tool like Postman to test the endpoints.
